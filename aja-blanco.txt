Stretch Questions

    1. What is the difference between an inline element and a block element?
Block-level elements take up all of the available space within their parent container.
Inline elements can exist within block-level elements. A paragraph would normally be a block-level element. A span would be an inline element and this could appear inside a paragraph.
    2. What happens when an element is positioned absolutely? 
an element that is absolutely positioned so other elements are positioned as if it did not exist. 
    3. How do I make an element take up only the amount of space it needs but also have the ability to give it a width?
